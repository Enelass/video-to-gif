# 🎬 video2gif-ruby

A Ruby script that converts any video file (MP4, MKV, AVI, MOV, WMV, etc.) to an optimized, lightweight GIF using FFmpeg.

## 📦 Features

- 🧠 Automatically detects supported video files in a folder
- 🧹 Scales videos to a max width of 1200px if needed
- 🎨 Reduces color depth for smaller file sizes
- 🕒 Converts at 3 frames per second (FPS)
- ✅ Pure Ruby — no external gems required
- 💻 Compatible with macOS, Linux, and Windows
